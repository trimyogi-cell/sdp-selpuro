Install Manual:

Chrome di PC:

  1. Klik ikon (tiga titik) di pojok kanan atas Chrome
  2. Pilih "Install aplikasi 'SPP SDN 1 Selopuro'"
  3. Klik Install

Chrome di Android:

  1. Klik ikon (tiga titik) di pojok kanan atas Chrome
  2. Pilih "Tambahkan ke Layar Utama"
  3. Klik "Tambahkan"

Safari di iPhone/iPad:

  1. Klik ikon (Share) di bawah
  2. Pilih "Tambahkan ke Layar Depan"
  3. Klik "Tambahkan"

Edge di PC:

  1. Klik ikon (tiga titik) di pojok kanan atas Edge
  2. Pilih "Aplikasi ini" > "Install situs ini sebagai aplikasi"
  3. Klik Install

Browser lainnya:

  1. Cari menu "Install", "Tambahkan ke Layar Utama", atau ikon di address bar
  2. Ikuti instruksi di layar
